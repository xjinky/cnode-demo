## 技术栈
Vue + Vue-Router + Vuex + Vuetify

## 项目安装
```
npm install    （推荐使用淘宝镜像加速）
```

### 启动服务
```
npm run serve  （vue-cli3.x项目用该命令启动服务）
```

### 项目打包
```
npm run build   （项目部署之前需要打包）
```

### 项目测试
```
npm run test     
```
